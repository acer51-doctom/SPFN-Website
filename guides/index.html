Troubleshooting - Wii U

I'm getting error code 118-XXXX when joining Splatoon matches

Error codes starting with 118 refer to the Peer to Peer connection. Either your connection or the connection of the host console is bad, resulting in errors like this. Ensure that your NAT type is either A or B (preferrably A) and try again.

I'm getting error code 616-XXXX when booting into the SPFN environment

SPFN requires you to be on the latest version of Aroma. Open the Aroma Updater while in the SPFN environment and update it.
